# Quick Guide: Creating External Vacantes

## For Coordinators

### Access the Feature

**Option 1: From Dashboard (Quick Access)**

1. Navigate to **Coordinador Dashboard**
2. Click the **"Crear Vacante Externa"** button in the header (next to "Enviar Correo Masivo")

**Option 2: From Vacantes List**

1. Navigate to **Coordinador Dashboard**
2. Go to **Vacantes Publicadas**
3. Click the **"Crear Vacante Externa"** button in the header

### Two Types of External Vacantes

#### 1. Image-Only Vacante (Flyer/Poster)

**Best for**: Job postings from external companies that provide a flyer or poster

**Required Fields**:

- External company name
- Job title
- Employment type (Full-time, Part-time, etc.)
- Work modality (Remote, On-site, Hybrid)
- Location
- State
- Job image (flyer/poster)

**Optional Fields**:

- Company contact email
- Company contact phone
- Career
- Number of positions
- Deadline

**Steps**:

1. Fill in external company name
2. Check ✓ "Vacante de solo imagen"
3. Fill basic job information
4. Upload the job poster/flyer
5. Click "Publicar Vacante Externa"

#### 2. Detailed External Vacante

**Best for**: Complete job descriptions for external companies

**Required Fields**:

- External company name
- Job title
- Employment type
- Work modality
- Location
- State
- Job summary
- Full description
- Responsibilities

**Optional Fields**:

- Company contact email
- Company contact phone
- Career
- Number of positions
- Salary range
- Job image
- Deadline

**Steps**:

1. Fill in external company information
2. Leave "Vacante de solo imagen" unchecked
3. Fill all job details in both tabs
4. Optionally upload an image
5. Click "Publicar Vacante Externa"

## Tips

### Image Requirements

- Formats: JPG, PNG, WebP, SVG
- Max size: 5MB
- Recommended: Clear, high-quality images

### Best Practices

- Always include company contact information when available
- Set realistic deadlines
- Use clear, descriptive job titles
- For image-only: Ensure the image contains all necessary information
- For detailed: Provide comprehensive job descriptions

### Validation

- Red dots on tabs indicate missing required fields
- Green dot on "Detalles" tab indicates image is selected
- Form won't submit until all required fields are filled

## What Happens After Creation

1. **Vacante is Published**: Immediately visible to students
2. **Appears in Lists**: Shows in all vacante listings with external company name
3. **Students Can Apply**: Normal application process
4. **Tracked by System**: Your coordinator account is recorded as creator
5. **Contact Info Displayed**: External company contact information shown to applicants

## Managing External Vacantes

- View all vacantes in "Vacantes Publicadas"
- External vacantes show the external company name
- Filter and search work normally
- Export to CSV includes external vacantes

## Common Scenarios

### Scenario 1: Company Sends Flyer

1. Company emails you a job flyer
2. Create image-only vacante
3. Upload the flyer
4. Add company contact info from email
5. Publish

### Scenario 2: Company Provides Details

1. Company sends job description
2. Create detailed vacante
3. Copy/paste job information
4. Add company details
5. Optionally add their logo/image
6. Publish

### Scenario 3: Job Fair Posting

1. Collect job info at fair
2. Create detailed vacante
3. Include booth/contact information
4. Set deadline for fair date
5. Publish

## Troubleshooting

**"El nombre de la empresa es requerido"**

- Fill in the external company name field

**"La imagen es requerida para vacantes de solo imagen"**

- Upload an image or uncheck "Vacante de solo imagen"

**"Error al publicar"**

- Check all required fields are filled
- Ensure image is under 5MB
- Verify you're logged in as coordinator

**Image won't upload**

- Check file format (JPG, PNG, WebP, SVG only)
- Ensure file is under 5MB
- Try a different image

## Need Help?

- Check the tabs for red dots indicating missing fields
- Hover over info icons (ℹ️) for field explanations
- Review error messages at the bottom of fields
- Contact system administrator if issues persist
